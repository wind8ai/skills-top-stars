## Inclusion Rules

### Source
- Only repos starred by wind8ai in the `Skills` list are included  
- URL: https://github.com/stars/wind8ai/lists/skills
- **Note**: The list has pagination — all pages must be checked

### Criteria
1. Based on wind8ai's stars
2. Public repository
3. Star ≥ 500 (unless exceptionally representative)
4. Main list capped at **24**, sorted by stars descending; rest go to candidates

### Sorting
- By total GitHub Stars, descending
- Stars shown as approximate magnitude (182k, 176k...)

### Candidates
- Capped at **18** (24 main + 18 candidates = 42)
- When full, lowest-star entries are dropped

### Maintenance
- **Description refinement**: wind8 periodically reviews and optimizes project descriptions
- **Author bios**: Added on first inclusion, revised as needed
- Star count changes → re-sort the list
- Newly starred repos enter candidates; overflow drops lowest
- Removed entries are annotated with reason and date

### Exclusion
- Repos with < 500 stars (unless exceptionally representative)
