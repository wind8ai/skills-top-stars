## Inclusion Criteria

### Source
- Only repositories starred by wind8Geek and tagged under the `Skills` list  
- URL: https://github.com/stars/wind8Geek/lists/skills
- **Note**: The list is paginated — all pages must be checked

### Requirements
1. Directly related to **AI Agent / Coding Agent Skills**
2. Public repository
3. Star count ≥ 500 (unless exceptionally noteworthy)
4. Top **24** by star count in the main list; others go to the candidate pool

### Sorting
- Descending by total GitHub stars
- Star counts shown as rounded magnitudes (137k, 135k…)

### Candidate Pool
- Max **18** entries (24 main + 18 candidate = 42 total)
- When full, the lowest-starred entry is dropped

### Maintenance
- **Descriptions & bios**: reviewed and refined periodically by 小乔 and 水镜
- Star count changes → update ranking
- New starred repos → candidate pool first; promote if a slot opens
- Removed entries → note reason and date next to the title: `❌ [YYYY-MM-DD] reason`

### Exclusions
- Repositories unrelated to AI / Coding Agent tooling
- Pure list repos with no substantive skills content
- Star count < 500 (unless exceptionally noteworthy)
