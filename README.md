## 收录规则

### 来源
- 仅收录 wind8Geek GitHub Stars 中标记为 `Skills` 列表的仓库  
- 地址：https://github.com/stars/wind8Geek/lists/skills
- **注意**：列表有分页，必须翻完所有页

### 收录条件
1. 以 wind8Geek 的 star 为准
2. 已公开（public repo）
3. Star ≥ 500（除非有特殊代表性）
4. 正式列表上限 **24 个**，按星数倒序；其余进候选池

### 排序
- 按 GitHub Star 总数倒序
- 星数仅展示量级（217k、205k...），不展示明细

### 候选池
- 上限 **24 个**（正式 24 + 候选 24 = 48）
- 超出时按星数淘汰最低者

### 更新维护
- **描述优化**：wind8 会不定期审视并优化各项目的描述内容，确保信息准确、表达清晰
- **作者简介**：首次收录时补充，后续维护时按需修正
- 发现页面星数变化 → 更新列表顺序
- 新 star 的项目进入候选池，候选池满时按星数淘汰最低者
- 剔除某个项目时在 README.md 对应条目**标题**旁注明原因和日期

### 不收录原则
- Star < 500 的项目（除非有特殊代表性）
