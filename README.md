- **注意**：列表有分页，必须翻完所有页

### 收录条件
1. 以 wind8ai 的 star 为准
2. 已公开（public repo）
3. Star ≥ 500（除非有特殊代表性）
4. 正式列表上限 **24 个**，按星数倒序；其余进候选池

### 排序
- 按 GitHub Star 总数倒序
- 星数仅展示量级（153k、149k...），不展示明细

### 候选池
- 上限 **18 个**（正式 24 + 候选 18 = 42）
- 超出时按星数淘汰最低者
